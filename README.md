# Space Delivery
## A lightweight game using pygame

*Space Delivery* is a 2D game in which the user is called to navigate
through dense asteroid clouds and reach a destination to deliver cargo.

It features random map generation for every attempt, system failures on collision with asteroids, as well as navigation and radar systems.

**Unfinished project, still in development**

## Installation
If not using prebuilt executables:
- Ensure you have Python 3.10 or above installed.
- Ensure you have `pip` installed. Otherwise, follow instructions at the [official PyPA site](https://pip.pypa.io/en/stable/installation/).
- Clone the repository in a folder of your choice (henceforth called `space_delivery/`)
- Install prerequisites: `pip install -r /path/to/space_delivery/requirements.txt`
- Run the `main.py` file.

Since the game is still in development, ensure you frequently check for updates to stay in the latest version.
